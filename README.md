# Pilbara Ridge Fleet Reliability and Maintenance Performance

## Project Overview
An end-to-end data analytics portfolio project focused on fleet reliability, maintenance performance, operational downtime and supply-chain delays within a simulated mining operation.

## Business Problem
Pilbara Ridge Mining Operations manages a 35-asset fleet across two 12-hour shifts. Operations leadership needs a clear view of repeat failures, downtime drivers, production loss, delayed parts and assets requiring maintenance intervention or replacement planning.

## Objectives
- Identify assets with the highest downtime and breakdown frequency.
- Analyse recurring failure categories across the fleet.
- Investigate whether parts delivery delays extend repair duration.
- Assess weather-related impacts on downtime, absenteeism and overtime.
- Examine the relationship between asset age, reliability and maintenance risk.
